WebAudio-Modem Demo

使用方法 (XModem Protocol)

  1. Initialize System をクリックしてシステムを初期化
  2. 🔄 XModem Loopback Test で内部テスト（推奨・マイク不要）
  3. 🎤 Enable Microphone でマイク権限を取得（実際の通信用）
  4. Using: Loopback/Microphone で入力ソースを切り替え
  5. Send/Receive でXModem通信を実行

XModem特徴: エラー検出・訂正、自動再送、確実なデータ転送
