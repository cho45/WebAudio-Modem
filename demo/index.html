WebAudio-Modem Demo

使用方法 (XModem Protocol)

  1. Initialize System をクリックしてシステムを初期化
  2. XModem Loopback Test で内部ループバックテスト
  3. Send Text/Image でスピーカーからXModem送信
  4. Start Receiving でマイクからのXModem受信開始

XModem特徴: エラー検出・訂正、自動再送、確実なデータ転送
